#Jarvis Voice Assistant 

what does it do ?
a voice-controlled AI assistant built in python that listens through the microphone and responds using text to speech

Features 
1) Voice recognition 
2) basic command handling 
3) Text to speech outpit 

How to run it

pip install -r requirements.txt
python main.py
